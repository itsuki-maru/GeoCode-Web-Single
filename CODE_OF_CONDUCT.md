# 行動規範

## 私たちの約束

私たちは、年齢、体格、障害の有無、民族性、性徴、性自認および性表現、経験年数、教育、社会経済的地位、国籍、容姿、人種、宗教、性的指向にかかわらず、誰もが参加しやすいコミュニティを目指します。

## 期待する行動

望ましい行動の例:

- 建設的で敬意あるフィードバックを行う
- 仕様や制約を前提に議論する
- 初学者にも分かる説明を心がける
- 問題の指摘と同時に改善案も示す

容認しない行動の例:

- 侮辱、中傷、嫌がらせ
- 差別的または性的な言動
- 個人情報の無断公開
- 脅迫や攻撃的な振る舞い

## 運用と対応

メンテナーは、この行動規範を明確にし、必要に応じて警告、コメント削除、Issue/PR のクローズ、参加制限などの対応を取ります。

## 適用範囲

この行動規範は、リポジトリ、Issue、Pull Request、Discussion、関連する公開コミュニケーションの場に適用されます。

## 報告方法

問題行動の報告は、リポジトリ管理者へ非公開で連絡してください。報告内容は可能な限り慎重に扱います。
